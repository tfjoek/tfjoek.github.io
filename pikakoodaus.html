Ohjelmointikielten esittely! Tässä muutama koodikieli jota on hyvä oppia ja kätetää paljo jep.

Python

Python on ohjelmointikieli, joka on monipuolinen ja sopii erityisesti verkkosovelluksiin.
Hauska fakta! Python nimi tulee tv-sarjasta Monty Python’s Flying Circus. Olen Python aloittelija.

JavaScript

JavaScript, usein lyhennettynä JS, on ohjelmointikieli, joka on yksi World Wide Webin ydintekniikoista HTML:n ja CSS:n rinnalla.
Hauska fakta! JavaScript-koodirivejä on olemassa yli 100 miljardia. Olen JavaScript ok.

Java

Java on korkean tason, luokkapohjainen, olio-ohjelmointikieli, joka on suunniteltu siten, että siinä on mahdollisimman vähän toteutusriippuvuutta.
Hauska fakta! Java oli alkuperäisesti nimeltään Oak. Olen Java tuntematon.

C++

C++ on korkeatasoinen, yleiskäyttöinen ohjelmointikieli, jonka on luonut tanskalainen tietojenkäsittelytieteilijä Bjarne Stroustrup.
Tylsä fakta! C ++ on yksi hallitsevista kielistä kaikenlaisten teknisten ja kaupallisten ohjelmistojen kehittämisessä. Olen C++ Tuntematon.

PHP

PHP on avoimen lähdekoodin palvelinpuolen komentosarjakieli, jota monet kehittäjät käyttävät verkkokehittäjille.
Hauska fakta! PHP: n maskotti on iso sininen norsu. Olen PHP tuntematon.
